# 🚀 Edgar (VaZ3D) - Cyberpunk Portfolio

A stunning, futuristic portfolio website showcasing Edgar's full-stack development skills with a cyberpunk aesthetic featuring pulsating neon buildings, digital rain, and smooth animations.

## ✨ Features

### 🎨 Visual Design
- **Cyberpunk Theme**: Dark background with neon pink, electric blue, and purple accents
- **Animated Background**: Pulsating neon buildings with digital rain effect
- **Smooth Animations**: 60fps performance with CSS animations and JavaScript interactions
- **Responsive Design**: Fully responsive across desktop, tablet, and mobile devices

### 🛠 Technical Features
- **Modern HTML5**: Semantic markup with accessibility features
- **CSS3 Animations**: Advanced keyframe animations and transitions
- **Vanilla JavaScript**: No framework dependencies, optimized performance
- **Mobile-First**: Responsive design with touch-friendly interactions
- **Performance Optimized**: Lazy loading, debounced scroll events, efficient animations

### 📱 Sections
1. **Hero Section**: Animated intro with cyber avatar and call-to-action buttons
2. **About**: Personal story and experience statistics
3. **Skills Matrix**: Interactive skill visualization with animated progress bars
4. **Experience**: Timeline showcasing IT development and WoW server experience
5. **Projects**: Featured project showcase (GhostBonus.com) with trophy badge
6. **Contact**: Contact form with social media links

## 🎯 Technologies Used

### Frontend
- **HTML5**: Semantic structure and accessibility
- **CSS3**: Advanced animations, Grid/Flexbox, CSS Variables
- **JavaScript (ES6+)**: Interactive features and smooth scrolling
- **Font Awesome**: Icons for skills and social links
- **Google Fonts**: Orbitron (futuristic) and Rajdhani (clean tech)

### Design Elements
- **Neon Color Palette**: Hot pink (#ff0080), Electric blue (#00ffff), Purple (#8a2be2)
- **Glass Morphism**: Backdrop blur effects and transparency
- **Particle System**: Dynamic background particles
- **Glitch Effects**: Cyberpunk-style text glitches
- **Holographic Elements**: Gradient animations and reflections

## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Local web server (optional, for development)

### Installation
1. Clone or download the project files
2. Open `index.html` in your web browser
3. For development, use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve .
   
   # Using PHP
   php -S localhost:8000
   ```

### File Structure
```
VAZED_Website/
├── index.html              # Main homepage
├── assets/
│   ├── css/
│   │   ├── style.css       # Main stylesheet
│   │   ├── animations.css  # Advanced animations
│   │   └── responsive.css  # Mobile responsiveness
│   ├── js/
│   │   └── main.js         # JavaScript functionality
│   └── images/             # Image assets (if any)
└── README.md               # Documentation
```

## 🎨 Customization

### Colors
Modify the CSS variables in `assets/css/style.css`:
```css
:root {
    --neon-pink: #ff0080;
    --neon-blue: #00ffff;
    --neon-purple: #8a2be2;
    --neon-green: #00ff41;
}
```

### Content
- Update personal information in `index.html`
- Modify skill levels in the skills section
- Add/remove projects in the projects grid
- Update social media links

### Animations
- Adjust animation speeds in `assets/css/animations.css`
- Modify particle count in `assets/js/main.js`
- Customize scroll-triggered animations
